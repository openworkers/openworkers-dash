Binding Setup

To use this storage config in your worker, add a binding in your environment settings:

  • Assets - Read-only access for serving static files
  • Storage - Full read/write access for object storage

Use the Config ID above when creating the binding.

Usage

Assets (read-only):

// Get a file
const file = await env.ASSETS.get('images/logo.png');

// List files
const list = await env.ASSETS.list({ prefix: 'images/' });

Storage (read/write):

// Get a file
const file = await env.STORAGE.get('uploads/file.pdf');

// Put a file
await env.STORAGE.put('uploads/file.pdf', data);

// Delete a file
await env.STORAGE.delete('uploads/file.pdf');

// List files
const list = await env.STORAGE.list({ prefix: 'uploads/' });
